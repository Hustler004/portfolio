Custom Git

Overview

I developed a custom implementation of core Git commands (init, cat-file, hash-object, ls-tree, write-tree, commit) entirely in JavaScript, aiming to replicate Git's functionality and storage mechanisms from scratch. This project provided a hands-on understanding of Git's inner workings, including content-addressable storage, hashing with SHA-1, and tree and commit object management. My implementation mirrors Git's unique approach to version control through efficient file handling and metadata preservation, showcasing my skills in low-level system design and JavaScript programming.
